
- URL: http://localhost:8080/ucptest-ws/webapi/books

- Get all:

curl -i -X GET http://localhost:8080/ucptest-ws/webapi/books

- Get all as xml: 

curl -H 'Accept:application/xml' -i -X GET http://localhost:8080/ucptest-ws/webapi/books

Get all books with ‘xy’ in their name:
curl -i -X GET http://localhost:8080/ucptest-ws/webapi/books/search/xy

Get book #10:
curl -i -X GET http://localhost:8080/ucptest-ws/webapi/books/10

Delete book #5:
curl -i -X DELETE http://localhost:8080/ucptest-ws/webapi/books/10

Add a new book:
curl -i -X POST -H 'Content-Type: application/json' -d '{"name": "New Book", "author": "Unknown 1"}' http://localhost:8080/ucptest-ws/webapi/books

Modify book #20: 
curl -i -X PUT -H 'Content-Type: application/json' -d '{"id": "20", "name": "New Book", "author": "Unknown 2"}' http://localhost:8080/ucptest-ws/webapi/books/20
